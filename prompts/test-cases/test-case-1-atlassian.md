# Test Case 1: Atlassian (Tech Company - Similar to SolarWinds)

## Test Parameters

**Company Name:** Atlassian

**Content/Narrative:**
Agile transformation success story presentation showing how a mid-sized enterprise (TechCorp Inc.) implemented Atlassian's suite (Jira, Confluence, Trello) and achieved:
- 40% faster sprint velocity
- 85% reduction in meeting time
- 95% team adoption rate within 3 months
- Improved cross-team collaboration

The presentation should cover:
1. The problem: Scattered tools, poor collaboration, slow delivery
2. The solution: Atlassian suite implementation strategy
3. The process: Phased rollout, training approach, adoption tactics
4. The results: Metrics, team feedback, business impact
5. The lessons learned: What worked, what didn't, recommendations

**Target Audience:** Enterprise IT leaders, CTOs, Engineering managers

**Presentation Goal:** Demonstrate Atlassian's value through real customer success story, inspire similar transformations, position Atlassian as essential for modern agile teams

**Desired Slide Count:** 12-14 slides

---

## Expected Quality Baseline (from SolarWinds Reference)

The output should match or exceed the SolarWinds deck quality on these dimensions:

### Brand Research Quality (10 points)
- Find exact Atlassian brand colors (should discover blue #0052CC, etc.)
- Identify official typography (likely Inter, Helvetica, or similar modern sans-serif)
- Extract brand personality (collaborative, modern, playful-but-professional)
- Discover recent brand refreshes or updates
- Document sources (Brandfetch, Atlassian design system, etc.)

### Visual Hierarchy Clarity (10 points)
- Every slide has PRIMARY (60-70%), SECONDARY (20-30%), TERTIARY (10%) defined
- Eye flow path explicitly described
- Visual weight percentages specified
- Focal point strategy clear
- Balance type justified

### Architecture Detail (10 points)
- Layout percentages specific (e.g., "65/35 split")
- Spacing measurements in px
- Grid structure defined
- Whitespace strategy with percentages
- All measurements provided (margins, padding, etc.)

### Specifications Completeness (10 points)
- All hex codes exact (not "blue" but "#0052CC")
- All font weights specific (not "bold" but "700")
- All sizes have units (px, pt, %)
- Icon styles consistently described
- Typography hierarchy complete

### Design System Quality (10 points)
- Comprehensive brand color documentation
- Complete typography system
- Icon guidelines clear
- Layout principles defined
- Accessibility standards included
- Production notes comprehensive
