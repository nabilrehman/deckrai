## Usage Tips

1. **For Midjourney:** Use --v 6 for latest model, --style raw for more literal interpretation, --s value controls stylization (lower = more literal)

2. **For DALL-E 3:** Be very descriptive about colors, shapes, and style. It handles gradients well.

3. **For Stable Diffusion:** Use negative prompts to avoid common issues. ControlNet with sketch can help guide composition.

4. **Iteration:** Generate multiple versions, then refine the best ones with slight prompt modifications.

5. **Export:** Request SVG or vector format when possible, or high-resolution PNG (at least 2000x2000px) for conversion to vector.
